# Node.js Rest server

This is a Basic Node.js Rest server, is built with TypeScript 

### Installation 🔧
```
$ npm install
```

## Getting started 🚀
```
$ tsc --watch
$ npm run dev
```

Open your browser and navigate to http://localhost:8000/
